ScreenBook is a full-stack web application for tracking movies and TV series. Users can search a live movie database, build watchlists, log films they've seen with personal ratings and reviews, track TV series episode-by-episode, and visualize their viewing habits through statistics and a calendar. Every user has a private account — data is completely isolated per login.
Key Features
•	User authentication — register, login, session management
•	Live movie/series search via OMDB API
•	Watchlist — save titles to watch later
•	Watched history — with star ratings (1–10), dates, and personal reviews
•	Favorites collection — star any title across all sections
•	Series Episode Tracker — season-by-season episode progress with 📍 position marker
•	Stats dashboard — genre breakdown, rating distribution, monthly activity chart
•	Watch calendar — visual day-by-day viewing history
•	Light/Dark theme toggle
•	Instant search suggestions dropdown with keyboard navigation
